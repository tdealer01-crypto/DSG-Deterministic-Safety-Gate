# Examples

This directory contains simple usage examples for the DSG beta.1 reference stack.

## Python example

```bash
pip install -e ./dsg-core[dev]
python examples/python/basic_usage.py
```

## JavaScript example

```bash
cd sdk/js
npm install
npm run build
node dist/index.js
```

You can also adapt `sdk/js/examples/basic.ts` for local integration work.
