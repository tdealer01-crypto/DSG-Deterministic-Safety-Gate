# RFC-0001: Deterministic Stability Governance (DSG) Protocol

Status: Draft  
Version: 0.1.0-beta.1  
Category: Experimental

## Abstract

The DSG protocol defines a deterministic governance layer for autonomous AI execution.
It combines policy invariant enforcement, harmonic stability analysis, and deterministic state transition gating.

## Canonical proof binding

Reference implementations bind the following fields into the proof hash:

- `version`
- `agent`
- `nonce`
- `signals`
- `state_next`
- `decision`
- `entropy`

## Decision states

- `ALLOW`
- `STABILIZE`
- `BLOCK`

## Request fields

{

## Security notes

- Replay protection is implemented in beta scope with nonce tracking.
- Production hardening requires atomic replay protection and distributed nonce storage.
- Ledger integrity is verified by recomputing chained record hashes.
